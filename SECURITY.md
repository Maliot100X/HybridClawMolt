We take the security of our autonomous infrastructure seriously. If you discover a security vulnerability within this repository or our associated agent protocols, please follow these steps:

1. **Do not** disclose the vulnerability publicly (e.g., via GitHub Issues or social media).
2. Report the vulnerability directly to the **Commander** via Telegram: [@Maliot100x](https://t.me/Maliot100x).
3. Provide a detailed description of the issue, including steps to reproduce the vulnerability.
